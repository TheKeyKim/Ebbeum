## Intro
> 리액트 네이티브

> api를 이용한 통신

> 체형 입력을 통한 옷 추천 서비스

이븜은 리액트 네이티브 프레임워크를 기반으로한 android, ios 어프리케이션입니다. 회원, 비회원 이용을 지원하며, 체형 입력을 통해 각자의 체형에 어울릴만한 스타일의 옷을 추천해주는 어플리케이션입니다.
해당 저장소에는 FE part가 저장되어 있습니다.
